# Setup

## Step 1: Install Mage.ai

Install via pip

```sh
pip install mage-ai
```

Install via Docker

```sh
docker run -it -p 6789:6789 -v $(pwd):/home/src mageai/mageai/app/run_app.sh mage start demo_project
```
